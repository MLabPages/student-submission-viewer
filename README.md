# 提出物連続確認ツール

学生の Word、PowerPoint、PDF を、元レイアウトに近いPDF表示で連続確認するWindows用ローカルツールです。

## 起動方法

`start-viewer.cmd` をダブルクリックします。ブラウザーに画面が開いたら「フォルダを選択」を押してください。

## デスクトップのショートカットから起動する（おすすめ）

毎回ツールのフォルダを開かなくても、デスクトップの専用アイコンから起動できます。

1. 初回だけ `create-desktop-shortcut.cmd` をダブルクリックします。
2. 「Desktop shortcut created successfully.」と表示されたら作成完了です。
3. 以後は、デスクトップの「提出物連続確認ツール」をダブルクリックします。

専用ショートカットから起動すると、黒いコマンド画面をほとんど表示せずブラウザーが開きます。ツールがすでに動いている場合は、二重起動せずブラウザー画面だけを開きます。

ショートカットはこのツールの保存場所を参照します。作成後にフォルダを移動した場合は、古いショートカットを削除して `create-desktop-shortcut.cmd` をもう一度実行してください。

手動で作成する場合は、`start-viewer.cmd` を右クリックし、「その他のオプションを確認」→「送る」→「デスクトップ（ショートカットを作成）」を選びます。

## 主な機能

- 現在の提出物を最優先でPDF化
- 次の3ファイルをバックグラウンドで先読み
- 変換済みPDFを再利用し、元ファイル更新時だけ再変換
- サブフォルダを含む提出物一覧と検索
- 前後のファイルへの連続移動
- 状態、点数、メモの自動保存
- 評価結果のCSV出力
- 元ファイルをWordまたはPowerPointで開く

## 対応形式

- Word: `.doc`, `.docx`
- PowerPoint: `.ppt`, `.pptx`
- PDF: `.pdf`

WordとPowerPointの変換には、このPCにインストールされているMicrosoft Officeを使います。提出物は外部へ送信されません。元ファイルは変更しません。

一時PDFと評価データは `%LOCALAPPDATA%\StudentSubmissionViewer` に保存されます。

## セキュリティ

- サーバーは `127.0.0.1` のみに接続し、同じPC以外からはアクセスできません。
- 外部Webページからの操作を拒否するHost・Origin検証とセキュリティヘッダーを設定しています。
- Office変換時はマクロを無効化し、ファイルを読み取り専用で開きます。
- CSV出力はExcelの数式として解釈される文字列を無害化します。
- 「元ファイルを開く」は通常のOfficeアプリで開くため、Officeの保護ビューや警告を解除する前に提出元を確認してください。

脆弱性と思われる問題は、公開のIssueでは学生情報や実ファイルを添付せず、再現手順だけを報告してください。
